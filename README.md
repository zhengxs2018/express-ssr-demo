## 启动项目

你需要安装 Node.js 的版本为 12+，克隆此仓库后运行:

``` bash
# 安装依赖
$ npm install

# 启动开发模式
$ npm run dev

# 启动调试模式，需要其他工具配合，如：vscode，Chrome devtools
$ npm run debug
```
